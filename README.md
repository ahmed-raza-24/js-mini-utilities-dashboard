# js-mini-utilities-dashboard
A mini JavaScript project demonstrating key JS concepts like arrays, objects, forEach, filter, reduce, and for...in loops with interactive UI.

## Features
- Sum calculator with step-by-step addition
- Filter students who passed (>= 40 marks)
- Calculate total price of product arrays
- forEach demo showing primitive value handling
- Object inspector: keys, values, key:value display, property count
- Store cumulative sums as an array

## Technologies
- HTML
- CSS
- JavaScript (vanilla)

## How to Run
1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Use the input fields and buttons to test each utility.
